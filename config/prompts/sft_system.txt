**CRITICAL: JSON Formatting Requirements**

When outputting JSON, you MUST follow these rules strictly:

1. **Escape all backslashes**: Any backslash (\) in a string must be escaped as double backslash (\\)

2. **Valid escape sequences only**: Only use these valid JSON escape sequences:
   - \" (quotation mark)

   - \r (carriage return)

   - \b (backspace)
